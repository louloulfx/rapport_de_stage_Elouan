1ère mission : Faire la partie front d’un site avec pour
modèles des maquettes précises.

Portfolio

Pour commencer mon stage j’ai tout d’abord découvert l’environnement de l’entreprise puis j’ai tout de suite été mis dans un projet avec un collège UI de Nantes ainsi que 3 collègues (2 UX et 1 UI) de Rennes. Le projet consistait à faire un portfolio avec les références de scalian ainsi que tous les sites que scalian a fait. J’ai dû de mon côté effectuer le site web. J’ai eu pour consigne au début de faire le site web en partant de rien et j’ai donc utilisé bootstrap.


Puis, afin d’être dans les bonnes conditions de construction de site web de l’entreprise j’ai dû apprendre à utiliser sass et gulp qui sont utiles afin de trier ces fichiers et de regrouper tous les fichiers de code dans un seul. En effet, grâce à gulp, tous les fichiers javascript sont instantanément placés dans un seul fichier qui prend toutes les lignes de codes : c’est ce fichier qui sera transmis au client ainsi que tous les autres fichiers compressés.

Maquette

Afin d’effectuer ce site web j’ai dû respecter un « style guide » ou j’ai dû mettre en place des polices d’écriture ainsi qu’une taille et des couleurs précises. J’ai également dû respecter les maquettes qui m’avaient été transmises pour le site web et donc j’avais pour consigne de rendre mon site identique aux maquettes. Après avoir fini mon site web, j’ai vérifié si celui-ci était compatible avec tous les formats d’écran ainsi qu’avec les principaux moteurs de recherche. Pour ce faire, je me suis aidé de sites qu’on m’a conseillés (Opquast Checklist, Box UK, et North Temple).
